# Kost Management

Kost Management is a property management application designed to help boarding house (kost) owners or managers manage their rooms in a structured and organized way.

## Features

- Room occupancy tracking
- Tenant management
- Monthly rent payment monitoring
- Dashboard overview of all rooms

## Tech Stack

- Rust
- Leptos
- Axum
- SQLx
- MySQL

## Goal

The main goal of this application is to simplify boarding house administration, reduce manual record-keeping, and improve accuracy in managing room and payment data.
